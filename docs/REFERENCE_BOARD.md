# Reference Board

External touchstones the design is calibrated against. Use these to check
new work by feel, not by literal imitation — the goal is the same register
(restraint, craft, confidence), not the same visual tricks.

## Tone references

- **Apple** — product-launch pacing: one idea per screen, generous negative
  space, motion that clarifies rather than decorates.
- **Nothing / Teenage Engineering** — industrial minimalism, monochrome
  confidence, typographic precision over ornament.
- **National Geographic / BBC Earth documentary titles** — scale, patience,
  and light used as the primary storytelling device (see the first-visit
  intro, "The Line," in [sprints/SPRINT_001_THE_LINE.md](sprints/SPRINT_001_THE_LINE.md)).
- **High-end print travel journalism** (Cereal, Kinfolk) — photography given
  room to breathe, minimal captioning, text as quiet support rather than
  decoration.

## What to avoid

- Generic "travel blog" visual language — bright accent colours, badge/pill
  UI, stock-photo gloss, busy card grids.
- Anything that reads as a template — the site's design should feel bespoke
  to this specific journey, not a themed skin.
- Decorative motion — any animation that doesn't clarify hierarchy, state,
  or story should be cut.

## Notes

This is a living mood reference, not a spec — update it when a new
touchstone genuinely shifts the direction, not for every small find.
